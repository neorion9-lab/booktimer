1. 북타이머란?
이 앱이 어떤 도움을 주는지 알아보세요

북타이머는 학급 독서 시간을 학생별로 기록하고, 학급 전체의 누적 독서 시간이 쌓일수록 자람나무가 성장하는 시각적 보상 시스템입니다. 학생들이 독서 버튼을 누르면 타이머가 시작되고, 읽은 시간이 자동으로 쌓입니다.

⏱️
개인별 타이머
학생 번호를 클릭하면 개인 독서 타이머가 시작·정지됩니다.
🌳
학급 자람나무
누적 독서 시간이 쌓일수록 나무가 새싹→사과나무로 성장합니다.
🏆
상위 학생 랭킹
독서 시간 상위 5명이 오른쪽 패널에 실시간으로 표시됩니다.
💾
자동 저장
기록이 브라우저에 자동 저장되어 새로고침해도 사라지지 않습니다.
💡 어떤 상황에서 쓰면 좋을까요? 아침 독서 시간, 독서 수업, 자유 독서 시간 등 학생들이 책을 읽는 모든 상황에서 활용하세요. 교실 TV나 대형 모니터에 띄워두면 학생들이 스스로 버튼을 누르고 독서 시간을 관리합니다.
🚀
2. 시작하기
처음 사용할 때 이것만 하면 됩니다
1
브라우저에서 index.html 파일 열기

파일을 더블클릭하거나, 크롬/엣지 브라우저에 드래그&드롭하여 엽니다. 인터넷 없이도 사용 가능합니다.

2
학생 수 설정하기 (기본값: 24명)

우측 상단 ⚙️ 버튼 클릭 → 👥 학생 수 변경에서 우리 반 인원 수를 입력하고 변경 버튼을 누릅니다.

3
제목 변경하기 (선택)

⚙️ 버튼 → ✏️ 제목 변경에서 반 이름이나 원하는 제목으로 바꿀 수 있습니다.

4
독서 시작! 학생 번호 버튼 클릭

독서를 시작하는 학생의 번호를 클릭하면 타이머가 시작됩니다. 다 읽었으면 다시 클릭하여 종료합니다.

✅ 기록은 자동으로 저장되므로 브라우저를 닫아도 다음 날 다시 열면 누적 기록이 그대로 남아있습니다.

3. 화면 구성 안내
각 영역이 어떤 역할을 하는지 알아보세요
상
📌 상단 헤더

왼쪽: 학급 제목 | 가운데: 현재 날짜·시간 (1초마다 자동 갱신) | 오른쪽: ⚙️ 설정 버튼

📚 왼쪽 패널 — 학생 버튼 그리드

학생 번호별 버튼이 표시됩니다. 클릭하면 타이머 시작/종료. 활동 중인 학생은 노란색으로 강조됩니다.

우

학급 전체 누적 독서 시간에 따른 나무 성장 단계와 상위 5명의 학생 랭킹이 표시됩니다.

하
💬 하단 푸터

독서 응원 문구가 랜덤으로 표시됩니다.

4. 학생 타이머 사용법
학생 버튼을 클릭하는 것만으로 모든 게 해결됩니다

학생 버튼은 아래와 같이 생겼습니다. 독서 전·후로 버튼을 한 번씩 누르면 됩니다.

5 ⏳ 오늘: 0초
누적: 0분
→
5 ⏳ 활동중 오늘: 12초
누적: 3분
→
5 ⏳ 오늘: 12초
누적: 3분

대기 중 → (클릭) → 활동 중 (노란색·모래시계 회전) → (클릭) → 종료

①
독서 시작: 학생 번호 한 번 클릭

버튼이 노란색으로 바뀌고 "활동중" 표시와 함께 모래시계가 회전합니다. 초가 실시간으로 올라갑니다.

②
독서 종료: 같은 번호 다시 클릭

버튼이 원래 색으로 돌아오고 오늘 읽은 시간과 누적 시간이 기록됩니다. 완료 메시지(팝업)가 나타납니다.

③
시간 표시 읽기

오늘: 오늘 하루 읽은 시간 (1분 이상이면 "N분 M초" 형태로 표시)
누적: 지금까지 모든 날의 총 읽은 시간 (분 단위)

💡 여러 학생 동시에 가능! 여러 학생을 동시에 활성화할 수 있습니다. 모든 학생이 함께 독서를 시작할 때는 각 학생이 자신의 번호를 누르도록 하면 됩니다.
⚠️ 브라우저를 닫으면 진행 중이던 타이머는 멈춥니다 다시 열었을 때 활성화 표시가 사라지는 것은 정상입니다. 누적 기록은 안전하게 저장되어 있습니다.
🌲
5. 자람나무 성장 단계
학급 전체 누적 독서 시간에 따라 나무가 성장합니다

오른쪽 패널의 나무 이모지는 모든 학생의 누적 독서 시간 합계를 기준으로 단계가 바뀝니다.

단계 이모지 이름 조건 (학급 누적 합계)
1단계 🌱 새싹 단계 0분 ~ 249분
2단계 🌿 잎사귀 나무 250분 ~ 499분
3단계 🌳 꽃 피는 나무 500분 ~ 749분
4단계 🌲 열매 맺는 나무 750분 ~ 999분
5단계 🏆 🍎 사과 열매 나무 1,000분 이상
🍎 사과 열매 단계 특별 기능! 학급 누적 독서 시간이 1,000분을 넘으면 나무 아래에 독서 시간이 가장 많은 상위 5명의 학생 번호가 사과 이모지와 함께 표시됩니다. 학생들에게 큰 동기부여가 됩니다!
⚙️
6. 설정 메뉴 안내
우측 상단 ⚙️ 버튼을 눌러 사이드바를 여세요
⏱️
퀵 타이머
10분 / 15분 / 20분 또는 직접 입력(1~60분)으로 학급 전체 독서 시간을 설정할 수 있습니다. (현재 카운트다운 기능은 준비 중)
🌳
성장 단계
현재 자람나무 단계와 학급 총 누적 독서 시간을 확인할 수 있습니다.
🤫
소음도 측정
마이크 권한을 허용하면 교실 소음 수준을 실시간으로 확인할 수 있습니다.
👥
학생 수 변경
1명~50명 사이에서 학생 수를 조정합니다. 설정값은 저장되어 다음에도 유지됩니다.
✏️
제목 변경
화면 상단의 제목을 우리 반 이름이나 원하는 문구로 바꿀 수 있습니다.
🔄
관리
모든 학생 활동 종료 — 현재 활동 중인 모든 학생의 타이머를 한 번에 끕니다.

모든 학생 기록 초기화 — 누적 독서 시간을 전부 0으로 리셋합니다. (복구 불가)
⚠️ 기록 초기화는 신중하게! "모든 학생 기록 초기화"는 모든 학생의 누적 시간이 영구적으로 삭제됩니다. 학기 말이나 새 학기 시작 전에만 사용하는 것을 권장합니다.
❓
7. 자주 묻는 질문
사용 중 궁금한 점들을 모았습니다
Q 학생이 버튼을 누르는 걸 깜빡했을 때는 어떡하나요?
기록되지 않은 시간은 직접 추가할 방법이 없습니다. 독서 시작 전에 함께 버튼을 누르는 루틴을 만들어 두는 것이 좋습니다. "책 펴기 전, 번호 누르기!" 같은 구호를 활용해 보세요.
Q 컴퓨터가 바뀌거나 다른 브라우저로 열면 기록이 사라지나요?
네, 기록은 현재 컴퓨터의 해당 브라우저에만 저장됩니다. 다른 기기나 다른 브라우저에서는 초기화된 상태로 보입니다. 항상 같은 컴퓨터와 같은 브라우저를 사용하세요.
Q 인터넷이 없어도 사용할 수 있나요?
네! 폰트(글꼴) 로딩 외에는 인터넷 연결 없이도 모든 기능이 정상 작동합니다. 오프라인 환경에서도 안심하고 사용하세요.
Q 오늘 시간과 누적 시간의 차이가 무엇인가요?
오늘: 현재 세션(브라우저를 연 이후)에서 읽은 시간입니다. 브라우저를 닫으면 초기화됩니다.
누적: 저장된 모든 독서 시간의 합계입니다. 브라우저를 닫아도 유지됩니다.
Q 전학생이 왔거나 번호가 바뀌었을 때는요?
학생 번호 기준으로 기록이 저장됩니다. 번호가 변경된 경우, 이전 번호의 누적 시간은 그 번호에 그대로 남습니다. 학기 초에 기록 초기화 후 새로 시작하는 것을 권장합니다.
Q 30명인데 24명까지만 보여요. 어떻게 하나요?
⚙️ 버튼 → 👥 학생 수 변경에서 원하는 인원수를 입력하면 됩니다. 최대 50명까지 설정 가능합니다.
